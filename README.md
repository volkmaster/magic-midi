# magic-midi

> Magic Midi Music Recorder.

## Commands

```bash
# install dependencies
npm install

# run server
npm run server

# build and watch in development mode
npm run dev

# build for production
npm run build

# deploy to heroku
heroku login
git add .
git commit -m "comment"
git push heroku master
```
